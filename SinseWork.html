Исследование в области квантовых вычислений: новые алгоритмы факторизации

Авторы: Иванов Иван Иванович, Петров Петр Петрович

Научный руководитель: д.ф.-м.н., проф. Сидоров С.С.

Кафедра: Вычислительной техники

Год публикации: 2024

Тип работы: Научная статья

Аннотация

В данной работе представлен новый алгоритм квантовой факторизации, основанный на модифицированном варианте алгоритма Шора. Предложенный метод позволяет сократить количество необходимых кубитов на 30% по сравнению с классической реализацией при сохранении той же вероятности успешного нахождения делителей.

Экспериментальные результаты, полученные на симуляторе квантового компьютера IBM Qiskit, демонстрируют эффективность алгоритма для чисел длиной до 16 бит. Теоретический анализ показывает возможность масштабирования метода для чисел большей разрядности.

Работа имеет важное значение для развития постквантовой криптографии и оценки стойкости современных криптографических систем.

1. Введение

Квантовые вычисления представляют собой перспективное направление, способное коренным образом изменить подходы к решению сложных вычислительных задач. Одним из наиболее известных квантовых алгоритмов является алгоритм Шора, позволяющий эффективно решать задачу факторизации целых чисел.

2. Методология

В основе предложенного метода лежит комбинация трех ключевых идей:

  1. Оптимизация квантового преобразования Фурье
  2. Использование частично запутанных состояний
  3. Итеративная процедура уточнения результатов

3. Результаты

Эксперименты проводились на симуляторе с 20 кубитами. Для тестирования были выбраны полупростые числа длиной от 8 до 16 бит.

Число Классический алгоритм (кубиты) Наш алгоритм (кубиты) Ускорение
247 (8 бит) 15 10 33%
4033 (12 бит) 23 16 30%
53513 (16 бит) 31 22 29%

4. Заключение

Предложенный алгоритм демонстрирует существенное сокращение требований к квантовым ресурсам при сохранении точности факторизации. Дальнейшие исследования будут направлены на:

  • Оптимизацию процедуры инициализации состояний
  • Адаптацию алгоритма для гибридных квантово-классических систем
  • Применение метода для решения других задач теории чисел
