Разработка энергоэффективного алгоритма маршрутизации для IoT-устройств с ограниченными ресурсами

Авторы: Кузнецов Алексей Викторович, Соколова Елена Дмитриевна

Научный руководитель: д.т.н., проф. Громов П.А.

Кафедра: Информационных систем и технологий

Год публикации: 2024

Тип работы: Научная статья

Аннотация

В статье представлен новый энергоэффективный алгоритм маршрутизации EERA (Energy-Efficient Routing Algorithm) для сетей Интернета вещей, основанный на комбинации методов машинного обучения и адаптивной логики. Алгоритм оптимизирует энергопотребление устройств при сохранении качества обслуживания.

Эксперименты на тестовом стенде с 50 узлами показали увеличение времени работы устройств на 35% по сравнению с традиционными протоколами RPL и LOADng. Алгоритм демонстрирует устойчивость к изменениям топологии сети и адаптивность к различным сценариям нагрузки.

Разработанное решение может быть применено в системах умного города, промышленного IoT и мониторинга окружающей среды.

1. Введение

Сети Интернета вещей (IoT) сталкиваются с проблемой ограниченных энергетических ресурсов устройств. Существующие протоколы маршрутизации не всегда эффективно учитывают динамику изменения состояния узлов и качество каналов связи.

2. Методология

Алгоритм EERA основан на следующих принципах:

  1. Многофакторная оценка метрики маршрута (энергия, качество связи, загрузка)
  2. Адаптивный механизм принятия решений на основе Q-обучения
  3. Децентрализованное управление таблицами маршрутизации
  4. Оптимизированный механизм обновления маршрутов

// Псевдокод алгоритма EERA

function selectRoute(node):

  routes = getAvailableRoutes(node)

  for route in routes:

    score = calculateRouteScore(route)

  bestRoute = selectRouteWithMaxScore(routes)

  updateLearningModel(bestRoute)

  return bestRoute

3. Результаты

Сравнительные тесты проводились на эмуляторе Cooja для платформы Contiki-NG. В качестве метрик использовались время жизни сети, латентность и процент доставленных пакетов.

Параметр RPL LOADng EERA (наш)
Время жизни сети (часы) 72.3 68.5 97.8
Средняя латентность (мс) 145 162 128
Доставка пакетов (%) 92.1 89.7 95.3
Накладные расходы (пакеты/час) 420 380 310

4. Заключение

Разработанный алгоритм EERA демонстрирует значительное преимущество по ключевым показателям энергоэффективности. Перспективы дальнейших исследований включают:

  • Интеграцию с протоколами безопасности IoT
  • Адаптацию для гетерогенных сетей
  • Реализацию на реальном оборудовании
  • Оптимизацию для 5G/6G сетей
