Оптимизация методов решения нелинейных дифференциальных уравнений в частных производных

Авторы: Смирнова Анна Владимировна, Козлов Дмитрий Сергеевич

Научный руководитель: д.ф.-м.н., проф. Васильев В.Г.

Кафедра: Прикладной математики

Год публикации: 2024

Тип работы: Научная статья

Аннотация

В данной работе предложен новый численный метод решения нелинейных дифференциальных уравнений в частных производных типа Навье-Стокса, основанный на модификации метода конечных элементов с адаптивными сетками. Основное внимание уделено оптимизации вычислительных алгоритмов для уменьшения времени расчета при сохранении точности.

Разработанный метод позволяет сократить количество итераций на 25-40% по сравнению с традиционными подходами для широкого класса задач. Теоретические оценки подтверждены численными экспериментами для модельных задач гидродинамики.

Результаты работы могут быть применены в вычислительной гидродинамике, метеорологии и других областях, требующих решения сложных нелинейных уравнений.

1. Введение

Нелинейные дифференциальные уравнения в частных производных играют ключевую роль в математическом моделировании физических процессов. Несмотря на значительный прогресс в вычислительных методах, задача разработки эффективных алгоритмов их решения остается актуальной.

2. Методология

Предложенный метод включает следующие ключевые элементы:

  1. Адаптивное построение сетки на основе апостериорных оценок
  2. Итерационный процесс с автоматическим выбором шага
  3. Гибридную схему линеаризации нелинейных членов
  4. Оптимизированный решатель СЛАУ

3. Результаты

Метод протестирован на стандартных задачах: обтекание цилиндра, течение в каверне и задача Тейлора-Грина. Сравнение проводилось с классическими методами конечных элементов и конечных объемов.

Задача Число узлов Традиционный метод (итерации) Наш метод (итерации) Экономия
Обтекание цилиндра (Re=100) 12,543 1,245 872 30%
Течение в каверне (Re=400) 8,762 987 592 40%
Тейлора-Грина (Re=500) 15,432 1,532 1,102 28%

4. Заключение

Разработанный метод демонстрирует значительное преимущество по числу итераций при сравнимой точности. Перспективы дальнейших исследований включают:

  • Применение методов машинного обучения для оптимизации адаптивных сеток
  • Распараллеливание алгоритма для GPU
  • Расширение класса решаемых уравнений
